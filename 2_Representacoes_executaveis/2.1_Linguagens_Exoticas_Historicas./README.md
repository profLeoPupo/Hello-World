# Linguagens Exóticas e Históricas 🌟

Esta subpasta contém exemplos do "Hello World!" em **linguagens de programação exóticas e históricas**. Muitas dessas linguagens são raras, desafiadoras ou projetadas com características únicas. São uma ótima oportunidade para explorar como conceitos clássicos podem ser expressos de maneira criativa e, por vezes, desafiadora.

### Linguagens Contidas

1. **Whitespace.ws**: Uma linguagem esotérica que usa apenas espaços, tabs e quebras de linha.
2. **Assembly 8086.asm**: Código em Assembly 8086, uma das primeiras linguagens de baixo nível.
3. **COBOL.cbl**: Implementação em COBOL, usada principalmente para sistemas bancários e comerciais.
4. **Fortran.f90**: Exemplo em Fortran, uma das linguagens mais antigas e usadas em computação científica.
5. **APL.apl**: Código em APL, conhecida por sua notação matemática concisa.
6. **Brainfuck.bf**: Uma das linguagens mais minimalistas e difíceis de programar.
7. **Malbolge.mal**: Código em Malbolge, projetada para ser uma das linguagens mais difíceis de programar.
8. **Befunge.bf**: Exemplo na linguagem Befunge, que possui um modelo de execução multidimensional.
9. **INTERCAL.i**: Linguagem de programação irreverente, projetada para ser difícil de entender.
10. **LOLCODE.lol**: Linguagem de programação baseada em memes da internet.

### Como Executar

Essas linguagens podem exigir ambientes específicos para execução ou ferramentas especiais. Consulte a documentação específica em cada arquivo para detalhes sobre como compilar ou interpretar os códigos.
