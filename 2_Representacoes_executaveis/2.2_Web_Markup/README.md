# Web Markup 🌐

Esta subpasta contém exemplos de "Hello World!" em **linguagens de marcação da web**. Essas linguagens são usadas para estruturar o conteúdo da web de maneira eficaz e sem a necessidade de compilação ou interpretação complexa.

### Arquivo Contido

- **hello_world.html**: Um exemplo básico em HTML para exibir a mensagem "Hello World!" em uma página da web.

### Como Usar

1. Abra o arquivo `hello_world.html` em seu navegador web.
2. A página exibirá a mensagem "Hello World!" na tela.
