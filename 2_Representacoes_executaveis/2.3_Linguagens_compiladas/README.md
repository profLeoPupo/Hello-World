# Linguagens Compiladas 🔧

Esta subpasta contém exemplos do "Hello World!" em **linguagens compiladas**. Linguagens compiladas exigem que o código seja convertido em um arquivo executável antes de ser executado.

### Arquivos Contidos

- **hello_world.c**: Exemplo em C, uma linguagem de baixo nível amplamente usada em sistemas e software de alto desempenho.
- **hello-world.go**: Código em Go, desenvolvido pela Google para facilitar o desenvolvimento de sistemas distribuídos e de alta performance.
- **hello_world.java**: Exemplo em Java, uma das linguagens mais populares para o desenvolvimento de aplicações corporativas.
- **hello-world.rs**: Implementação em Rust, uma linguagem que foca em segurança e performance.
- **hello-world.swift**: Exemplo em Swift, a linguagem da Apple para desenvolvimento de aplicativos iOS e macOS.

### Como Compilar e Executar

1. **C**: Compile com `gcc hello_world.c -o hello_world` e execute com `./hello_world`.
2. **Go**: Compile com `go run hello-world.go`.
3. **Java**: Compile com `javac hello_world.java` e execute com `java hello_world`.
4. **Rust**: Compile com `cargo run`.
5. **Swift**: Compile com `swift hello-world.swift`.
