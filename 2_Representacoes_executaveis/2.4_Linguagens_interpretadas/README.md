# Linguagens Interpretadas 📝

Esta subpasta contém exemplos do "Hello World!" em **linguagens interpretadas**. Linguagens interpretadas são executadas diretamente por um interpretador, sem a necessidade de uma etapa de compilação prévia.

### Arquivos Contidos

- **hello_world.js**: Exemplo em JavaScript, utilizado principalmente para desenvolvimento web.
- **hello_world.php**: Código em PHP, frequentemente usado para desenvolvimento de websites dinâmicos.
- **hello_world.py**: Exemplo em Python, uma linguagem conhecida pela sua simplicidade e legibilidade.

### Como Executar

1. **JavaScript**: Execute o arquivo diretamente no navegador ou com o Node.js.
2. **PHP**: Execute o arquivo com o comando `php hello_world.php`.
3. **Python**: Execute com `python hello_world.py`.
