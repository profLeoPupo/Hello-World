# Linguagens de Consulta a Banco de Dados 📊

Esta subpasta contém exemplos do "Hello World!" em **linguagens de consulta a banco de dados**. Esses códigos demonstram como interagir com bancos de dados usando SQL, a linguagem padrão para gerenciamento de dados.

### Arquivo Contido

- **hello_world.sql**: Exemplo de código SQL para exibir "Hello World!" em um banco de dados.

### Como Executar

1. Execute o código SQL no seu banco de dados, utilizando qualquer sistema de gerenciamento de banco de dados (SGBD) que suporte SQL.
2. O comando exibirá a mensagem "Hello World!" no console ou interface do banco de dados.
