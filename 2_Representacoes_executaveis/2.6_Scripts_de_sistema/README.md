# Scripts de Sistema ⚙️

Esta subpasta contém exemplos do "Hello World!" em **scripts de sistema**. Scripts de sistema são usados para automatizar tarefas em sistemas operacionais e podem ser escritos em diferentes linguagens de script.

### Arquivos Contidos

- **hello_world.bat**: Exemplo de script em Batch para Windows.
- **hello_world.ps1**: Script em PowerShell para automação em Windows.
- **hello_world.sh**: Exemplo de script em Shell para sistemas Unix/Linux.

### Como Executar

1. **Batch**: Execute o arquivo `hello_world.bat` clicando duas vezes sobre ele.
2. **PowerShell**: Execute `hello_world.ps1` no PowerShell.
3. **Shell**: Execute `hello_world.sh` no terminal (Linux/Mac) com `bash hello_world.sh`.
