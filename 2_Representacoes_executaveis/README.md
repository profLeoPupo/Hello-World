# Representações Executáveis 🖥️💻

Esta pasta contém diversas implementações **executáveis** do algoritmo "Hello World!" em várias linguagens de programação. O objetivo é fornecer exemplos práticos do código em diferentes contextos, permitindo que você experimente como o algoritmo é expresso em várias linguagens. 🌍🚀

## Estrutura

As representações aqui contidas são organizadas em diferentes subcategorias, para facilitar o acesso a exemplos específicos. Explore as linguagens, compilenos ou interprete-os para visualizar como o código executa o tradicional "Hello World!".

### Arquivos Disponíveis

#### 2.1 Linguagens Exóticas e Históricas 🌟

Aqui você encontrará exemplos em linguagens raras e históricas, conhecidas por sua complexidade ou seu uso único.

- **2.6.10_Whitespace.ws**: Exemplo na linguagem Whitespace, que usa apenas espaços, tabs e quebras de linha para codificação.
- **2.6.1_Assembly_8086.asm**: Código em Assembly 8086, uma das primeiras linguagens de baixo nível.
- **2.6.2_COBOL.cbl**: Implementação no COBOL, uma das linguagens mais antigas e usadas para programação de negócios.
- **2.6.3_Fortran.f90**: Exemplo em Fortran, utilizado principalmente em cálculos científicos e engenharia.
- **2.6.4_APL.apl**: Código na linguagem APL, conhecida por sua notação matemática concisa.
- **2.6.5_Brainfuck.bf**: Um exemplo no Brainfuck, uma linguagem esotérica projetada para ser extremamente minimalista.
- **2.6.6_Malbolge.mal**: Código em Malbolge, uma das linguagens mais difíceis de programar.
- **2.6.7_Befunge.bf**: Exemplo na linguagem Befunge, com seu paradigma multidimensional.
- **2.6.8_INTERCAL.i**: Implementação em INTERCAL, famosa pela sua sintaxe confusa e irreverente.
- **2.6.9_LOLCODE.lol**: Exemplo no LOLCODE, projetado para parecer com "linguagem de internet" ou memes.

#### 2.2 Web Markup 🌐

Exemplo de "Hello World!" em uma linguagem de marcação da web.

- **hello_world.html**: Um simples arquivo HTML que exibe "Hello World!" em uma página web.

#### 2.3 Linguagens Compiladas 🔧

Exemplos de código em linguagens compiladas, que precisam ser processadas por um compilador antes de serem executadas.

- **hello_world.c**: Exemplo em C, uma das linguagens mais populares de baixo nível.
- **hello-world.go**: Implementação em Go, uma linguagem desenvolvida pela Google com foco em performance e simplicidade.
- **hello_world.java**: Código em Java, uma linguagem amplamente usada para desenvolvimento de aplicativos.
- **hello-world.rs**: Exemplo em Rust, uma linguagem que foca em segurança e performance.
- **hello-world.swift**: Implementação em Swift, linguagem desenvolvida pela Apple para aplicativos iOS e macOS.

#### 2.4 Linguagens Interpretadas 📝

Aqui estão exemplos de linguagens interpretadas, que são executadas diretamente por um interpretador.

- **hello_world.js**: Exemplo em JavaScript, amplamente usado para desenvolvimento web.
- **hello_world.php**: Código em PHP, frequentemente usado para desenvolvimento de sites dinâmicos.
- **hello_world.py**: Exemplo na linguagem Python, famosa pela sua simplicidade e legibilidade.

#### 2.5 Linguagens de Consulta a Banco de Dados 📊

Exemplo de implementação do "Hello World!" em uma linguagem usada para interagir com bancos de dados.

- **hello_world.sql**: Código SQL para exibir uma mensagem "Hello World!" em um banco de dados.

#### 2.6 Scripts de Sistema ⚙️

Exemplos de scripts usados para automação de tarefas em sistemas operacionais.

- **hello_world.bat**: Script em Batch, usado em sistemas Windows.
- **hello_world.ps1**: Script em PowerShell, uma linguagem de script para automação de tarefas no Windows.
- **hello_world.sh**: Script em Shell, utilizado em sistemas Unix/Linux para automação.

### Como Usar

1. **Linguagens Exóticas e Históricas**: Estes códigos podem exigir ambientes específicos ou ferramentas para compilar e executar. Consulte o README em cada subpasta para mais detalhes.
2. **Web Markup**: Abra o arquivo `hello_world.html` em um navegador web para visualizar a execução.
3. **Linguagens Compiladas**: Compile os códigos com seus respectivos compiladores e execute-os para ver o "Hello World!" na saída.
4. **Linguagens Interpretadas**: Execute diretamente no terminal ou ambiente de desenvolvimento apropriado para ver o resultado.
5. **Consultas a Banco de Dados**: Execute o código SQL em um banco de dados que suporte o comando para ver o retorno.
6. **Scripts de Sistema**: Execute os scripts em seu terminal ou ambiente de script apropriado (Windows PowerShell, Bash, etc.).
