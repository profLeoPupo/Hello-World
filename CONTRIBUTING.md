# Como Contribuir 🤝

**Obrigado por considerar contribuir para o projeto Hello World!** 🙏

Este repositório é dedicado ao aprendizado colaborativo. Se você deseja contribuir, há várias formas de participar. Seja adicionando novos exemplos de código, melhorando a documentação, corrigindo erros ou sugerindo melhorias, sua colaboração é muito bem-vinda!

Aqui estão as etapas para contribuir de maneira eficaz:

## 🔄 Passos para Contribuir

### 1. Faça um Fork do Repositório 🍴
- Clique no botão **"Fork"** no canto superior direito desta página para criar uma cópia do repositório em sua conta.
- O link do repositório é [GitHub - Hello World](https://github.com/profLeoPupo/Hello-World).

### 2. Clone o Repositório para o Seu Computador 🖥️
- Após fazer o fork, clone o repositório para sua máquina local com o seguinte comando:

    ```bash
    git clone https://github.com/seu-usuario/Hello-World.git
    ```

### 3. Crie uma Nova Branch para Suas Alterações 🌱
- Antes de começar a modificar, crie uma nova branch para trabalhar de forma isolada:

    ```bash
    git checkout -b nome-da-sua-branch
    ```

### 4. Faça suas Alterações ✏️
- Agora você pode começar a fazer suas mudanças! Adicione novos exemplos, corrija erros, ou melhore a documentação.
- Não se esqueça de seguir o estilo de codificação e formatação do projeto.

### 5. Commit suas Alterações 📝
- Após fazer as mudanças, registre suas alterações com um **commit**:

    ```bash
    git add .
    git commit -m "Descrição clara do que foi feito"
    ```

### 6. Envie suas Alterações para o GitHub ⬆️
- Faça o **push** para enviar suas alterações para sua branch no GitHub:

    ```bash
    git push origin nome-da-sua-branch
    ```

### 7. Crie um Pull Request 🔀
- Após o push, vá até o repositório original em [GitHub - Hello World](https://github.com/profLeoPupo/Hello-World) e clique em **"Compare & Pull Request"**.
- Descreva claramente as mudanças que você fez e por que acha que elas são importantes.

## 🧑‍💻 O Que Podemos Aceitar?

Estamos interessados em **melhorias educacionais**! Aqui estão algumas sugestões de contribuição:

- **Novas linguagens ou representações do "Hello World!"** 🌍💻
- **Correções de erros** (ortografia, formatação, erros no código) 🐛
- **Melhorias na documentação** 📚
- **Sugestões de novos tutoriais ou exemplos** 💡

## ⚖️ Código de Conduta

Por favor, seja respeitoso com todos os colaboradores. Nosso objetivo é criar um ambiente positivo e acolhedor. O respeito é fundamental!

- Seja **amigável** e **colaborativo**.
- **Respeite as ideias** e sugestões dos outros.
- Evite qualquer tipo de **discussão negativa ou hostil**.
